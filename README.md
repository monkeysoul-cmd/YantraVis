## 🎯 The Problem
Historically, there has been a lack of unified digital repositories for ancient astronomical instruments (Yantras), leading to the loss of ancient knowledge, low awareness, and educational gaps. Without computational tools, recalculating alignment and accuracy for specific locations across India remains incredibly difficult, preserving only location-specific designs rather than universal solutions.

## 💡 Our Solution: YantraVis
A comprehensive software that, for any latitude/longitude in India, produces parametric construction dimensions for listed yantras (Samrat, Rama, Jai-Prakash, Rashivalaya, etc.).

- **Parametric Construction Dimensions**: Produces mathematically accurate dimensions for listed yantras based on geolocation.
- **3D Visualization**: Visualizes the instrument in 3D and simulates solar operation (shadow movement).
- **CAD Export & Installation**: Exports CAD/3D files and provides orientation/installation guidance (true north, magnetic declination).
- **AI Analysis**: Optionally produces BOM/cost & accuracy estimates.

### Key Features
- **Data Validation**: Ensures accuracy and reliability of data.
- **3D Visualization**: Improves data representation and understanding.
- **Astronomy Libraries**: Provides access to valuable astronomical resources.
- **Parametric Modelling**: Enhances design efficiency and precision.
- **Output Flexibility**: Accommodates diverse user needs and preferences.

## 🚀 Application Flow
1. **User Input**: Users enter their latitude and longitude coordinates.
2. **Instrument Selection**: Choose from ancient instruments like Samrat Yantra, Rama Yantra, Jai-Prakash, etc.
3. **Core Computation**: Web App dynamically processes location data.
4. **Data Generation**: 
   - **Parametric 3D preview**: Interactive 3D preview generated in real-time.
   - **Downloadable files**: DXF, STL, and glTF files available for 3D printing and CAD.
   - **Orientation guide**: True north angle and magnetic declination correction guide.
   - **Accuracy estimate**: AI-generated estimation of accuracy, readout resolution, and building materials.
